Paper Based Design

I plan to create a simple website with the navigation running along the top of the page. I will create a banner to run along the top of the page that will be used on every page. This will help to imitate the look and feel of a national news site.

For the sites navigation I plan to have each link open a set of sub links below each link. These will display on a second navigation menu below the original link to make a sub link menu. The pages content will then display below the navigation menus.
