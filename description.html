Description of the Project

I intend to create a weather report using flash. It will cover a three day period, describe several types of weather and will be easy to navigate through the use of buttons.

I will also create a news report using original video that cuts between several different scenes.

The video and flash project will be accompanied by a website that describes its creative design and imitates a national news website.
